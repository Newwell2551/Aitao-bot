# คู่มือตั้งค่าโปรเจกต์ Aitao Bot (สำหรับทีม Web Dashboard)

> เอกสารนี้สำหรับคนที่จะช่วยพัฒนาหน้าเว็บของ Aitao Bot
> ทำตามทีละขั้น อย่าข้ามนะคะ

---

## ขั้นที่ 1: ติดตั้งโปรแกรมที่จำเป็น

### 1.1 Visual Studio Code (โปรแกรมเขียนโค้ด)
1. เปิดเบราว์เซอร์ไปที่ https://code.visualstudio.com
2. กดปุ่ม **Download for Windows**
3. เปิดไฟล์ที่โหลดมา แล้วกด Next ไปเรื่อยๆ จนกด Install
4. รอติดตั้งเสร็จ แล้วกด Finish

### 1.2 Node.js (ตัวรันโค้ด JavaScript)
1. ไปที่ https://nodejs.org
2. กดดาวน์โหลดปุ่มซ้าย ที่เขียนว่า **LTS** (ไม่ใช่ Current)
3. เปิดไฟล์ที่โหลดมา กด Next ไปเรื่อยๆ จนกด Install
4. รอติดตั้งเสร็จ

**ตรวจสอบว่าติดตั้งสำเร็จ:**
- กด `Windows + R` พิมพ์ `cmd` กด Enter
- พิมพ์ `node -v` แล้วกด Enter
- ถ้าขึ้นเลขเวอร์ชัน เช่น `v22.0.0` แสดงว่าสำเร็จค่ะ

### 1.3 Git (ระบบจัดการโค้ดร่วมกัน)
1. ไปที่ https://git-scm.com/download/win
2. กดดาวน์โหลดอัตโนมัติ (ถ้าไม่โหลดเองให้กดลิงก์ที่หน้านั้น)
3. เปิดไฟล์ที่โหลดมา กด Next ไปเรื่อยๆ **ไม่ต้องเปลี่ยนอะไร** จนกด Install
4. รอติดตั้งเสร็จ กด Finish

---

## ขั้นที่ 2: ตั้งค่า GitHub

GitHub คือที่เก็บโค้ดกลางที่ทีมใช้ร่วมกันค่ะ

### 2.1 สมัครบัญชี GitHub (ถ้ายังไม่มี)
1. ไปที่ https://github.com
2. กด **Sign up** ตั้งชื่อผู้ใช้ อีเมล และรหัสผ่าน

### 2.2 ตั้งค่า Git ในเครื่อง
เปิด cmd แล้วพิมพ์ทีละบรรทัด (เปลี่ยนชื่อและอีเมลให้ตรงกับ GitHub):
```
git config --global user.name "ชื่อของคุณ"
git config --global user.email "อีเมลที่ใช้สมัคร GitHub"
```

### 2.3 รอรับคำเชิญเข้า repo
เจ้าของโปรเจกต์จะส่งคำเชิญมาทางอีเมล GitHub — กดยืนยันในอีเมลนั้นเลยค่ะ

---

## ขั้นที่ 3: โหลดโค้ดมาที่เครื่อง

1. เปิด VS Code
2. กด `Ctrl + Shift + P` แล้วพิมพ์ `Git Clone` กด Enter
3. วาง URL นี้: `https://github.com/Newwell2551/Aitao-bot.git`
4. เลือกโฟลเดอร์ที่จะเก็บโปรเจกต์ (เช่น Desktop)
5. กด Open เมื่อขึ้นถาม

**จากนี้โฟลเดอร์ "Aitao-bot" จะอยู่ในเครื่องคุณแล้วค่ะ**

---

## ขั้นที่ 4: สร้าง Branch สำหรับงาน Web

Branch คือ "สาขา" ที่แยกการทำงานออกจากโค้ดหลัก ไม่กระทบกัน

1. ใน VS Code กดที่ชื่อ `main` มุมซ้ายล่างของหน้าจอ
2. เลือก **Create new branch**
3. ตั้งชื่อว่า `web-dashboard`
4. กด Enter

**จากนี้ทุกอย่างที่ทำจะอยู่ใน branch `web-dashboard` ไม่กระทบ bot หลักค่ะ**

---

## ขั้นที่ 5: ทำความเข้าใจโครงสร้างโปรเจกต์

```
Aitao-bot/
├── index.js              ← ตัวหลักของ Discord bot (ไม่ต้องแตะ)
├── commands/             ← คำสั่งต่างๆ ของ bot (ไม่ต้องแตะ)
├── utils/                ← ฟังก์ชันช่วยเหลือต่างๆ (ไม่ต้องแตะ)
├── package.json          ← รายชื่อ library ที่ใช้
└── web/                  ← โฟลเดอร์ที่คุณจะสร้างและทำงานในนี้ค่ะ
    ├── index.html        ← หน้าแรก (landing page)
    ├── dashboard.html    ← หน้า dashboard
    ├── pricing.html      ← หน้าราคา
    └── style.css         ← สไตล์ทั้งหมด
```

**สรุปง่ายๆ**: คุณทำงานใน **โฟลเดอร์ `web/` เท่านั้น** ไม่ต้องยุ่งกับไฟล์อื่นเลยค่ะ

---

## ขั้นที่ 6: ตั้งค่า Claude สำหรับทำงาน

### 6.1 สร้าง Claude Project ใหม่
1. เปิด claude.ai
2. กด **Projects** → **New Project**
3. ตั้งชื่อว่า "Aitao Bot - Web Dashboard"
4. ใส่ข้อความนี้ใน Project instructions:

```
โปรเจกต์: เว็บไซต์และ Dashboard สำหรับ Aitao Bot (Discord bot)
ภาษาที่ใช้ในโค้ด: HTML, CSS, JavaScript
ฉันเป็นมือใหม่ด้านการเขียนโค้ด ต้องการคำอธิบายทุกขั้นตอนแบบละเอียด
อธิบายว่าโค้ดแต่ละส่วนทำอะไร ไม่ข้ามการอธิบาย
```

### 6.2 อัปไฟล์เข้า Project knowledge
อัปไฟล์เหล่านี้เพื่อให้ Claude รู้จักโปรเจกต์:
- `package.json`
- `utils/buildMessageFromSchema.js` (engine หลักของ bot ที่เว็บจะเรียกใช้)

---

## ขั้นที่ 7: วิธี Push โค้ดขึ้น GitHub

เมื่อทำงานเสร็จในแต่ละวัน ให้ push ขึ้น GitHub เพื่อให้ทีมเห็นความคืบหน้า

เปิด Terminal ใน VS Code (กด `Ctrl + `` `) แล้วพิมพ์:
```bash
git add .
git commit -m "อธิบายสั้นๆ ว่าทำอะไรวันนี้"
git push
```

ตัวอย่าง commit message:
- `"ทำ landing page เสร็จแล้ว"`
- `"เพิ่มหน้า pricing"`
- `"แก้สีปุ่ม header"`

---

## ข้อควรจำ

| ✅ ทำได้ | ❌ ห้ามทำ |
|---------|---------|
| แก้ไขหรือสร้างไฟล์ใน `web/` | แก้ไขไฟล์นอก `web/` |
| Push ขึ้น branch `web-dashboard` | Push ขึ้น branch `main` โดยตรง |
| ถาม Claude ได้ตลอด | สร้างหรือลบไฟล์ `.env` |
